# 💰 Pocket Money Manager

Pocket Money Manager is a modern neon-themed finance tracking web application built using Flask, HTML, CSS, and JavaScript.

This app helps parents monitor allowance spending while giving children a simple and interactive way to manage their pocket money.

## 🚀 Features

✨ Attractive Neon UI Design
🪙 Animated Coin Toss Loading Screen
👨 Parent Dashboard
🧒 Child Dashboard
🔐 Login & Logout System
💰 Allowance Tracking
📉 Expense Monitoring
📑 Transaction History
🏆 Streaks & Gold Badges
⚡ Smooth Navigation Flow
📱 Mobile Friendly Layout

## 🛠️ Technologies Used

* Python (Flask)
* HTML5
* CSS3
* JavaScript
* SQLite (Future Upgrade)
